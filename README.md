# Sistem Manajemen TPA

Website untuk mengelola data santri, kelas, hafalan, dan absensi pada Taman Pendidikan Al-Qur'an (TPA).

## Fitur

* Login Admin
* Logout
* Dashboard Statistik
* CRUD Data Santri
* Data Kelas
* Data Hafalan
* Data Absensi

## Teknologi

* HTML
* CSS
* Bootstrap 5
* JavaScript
* PHP Native
* MySQL

## Struktur Database

* users
* santri
* kelas
* hafalan
* absensi
* log_aktivitas

## Cara Menjalankan

1. Jalankan Apache dan MySQL pada XAMPP.
2. Import file database.sql ke phpMyAdmin.
3. Simpan folder project ke dalam htdocs.
4. Akses melalui browser:

http://localhost/tpa-management
